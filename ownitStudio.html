Google Sheet Intake Setup
Replace Formspree with a free Google Apps Script endpoint — your form design stays exactly the same.
  1. Create a new Google Sheet. Name it "OwnIt Intake Submissions".
  2. Click Extensions → Apps Script. Delete any existing code and paste this:
function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = JSON.parse(e.postData.contents); if (sheet.getLastRow() === 0) { sheet.appendRow(Object.keys(data)); } sheet.appendRow(Object.values(data)); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
  1. Click Deploy → New Deployment. Type: Web app. Execute as: Me. Access: Anyone. Click Deploy and copy the URL.
  2. Paste that URL below and click Save — it will replace the Formspree endpoint in your intake form instructions.
  3. In your intake form HTML, replace the Formspree action URL with this new URL. You can then cancel Formspree.
